# Predictive Maintenance Machine Learning

This project builds a machine learning model to predict industrial machine failures using operational sensor data.

## Dataset
AI4I 2020 Predictive Maintenance Dataset

## Tools Used
Python  
Pandas  
NumPy  
Scikit-learn  
Matplotlib  

## Project Steps
1. Exploratory Data Analysis
2. Outlier Detection
3. Feature Engineering
4. Correlation Analysis
5. Logistic Regression Model
6. Random Forest Model
7. Model Evaluation

## Results
Accuracy: 99.9%  
Failure Detection Recall: 97%

## Key Predictive Features
Tool wear  
Mechanical power (Torque × RPM)  
Torque  
Rotational speed
